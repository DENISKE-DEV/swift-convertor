User Guide

This application is designed to assist users in converting SWIFT messages of type pacs.008 into pacs.004, specifically for cases involving the return of funds that were incorrectly received on one of our accounts.

Purpose

The tool simplifies the process of generating a compliant pacs.004 message from a pacs.008 input, ensuring accurate and efficient handling of return transactions.

How to Use the Application

  • Copy the full XML content of the pacs.008 message.
  • Paste it into the Input message XML pacs.008 field.
  • Click the Convert button to generate the pacs.004 message.
  • The converted message will appear in the Output section.

Notes

  • Ensure the pacs.008 message is complete and correctly formatted before conversion.
  • This tool is intended for internal use by authorized personnel.
  • Contact the Back Office support team for any issues.
